# ARIA Labels

## Beschreibende Attribute

### Was ist der Unterschied zwischen `aria-label` and `title`?

Manchmal müssen Elemente mit labels oder titles näher beschrieben werden. Zum Beispiel wenn ein Button nur mittels Icon dargestellt werden soll.

Manche Screenreader (NVDA v2024.1) lesen beides vor, was zu Dopplungen führt. Der Vorteil des `title` ist, dass ein Popup on hover erscheint. Aber auch nur für Desktop bzw. Maus Nutzer. Ein individuelles Abwägen ist hier erforderlich.
